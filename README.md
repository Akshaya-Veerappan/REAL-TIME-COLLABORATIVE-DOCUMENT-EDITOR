# 📝 Real-Time Collaborative Document Editor

A backend-based collaborative document editor that allows multiple users to work on documents in real time.

## 🚀 Features

- Real-time document updates  
- Backend API for handling user requests  
- Basic collaboration functionality  

## 🛠 Tech Stack

- Node.js  
- Express.js  
- JavaScript  

## 📂 Project Structure

- app.js – Main server file  
- routes.js – API routes  
- db.js – Database connection  
- message.js – Messaging logic  

## ⚙️ How to Run Locally

```bash
git clone https://github.com/Akshaya-Veerappan/REAL-TIME-COLLABORATIVE-DOCUMENT-EDITOR.git
cd REAL-TIME-COLLABORATIVE-DOCUMENT-EDITOR
npm install
node app.js
